Every Problem Faced With PyInstaller

Overview

Pyinstaller is a bundler that bundles the python code and all its dependencies in the project to an executable rather than compiling it. We decided to use it because it converts python code directly to an executable without the C-code generation Nuitka deals with.

Py2Exe compilation

Py2Exe is a tool converting Python scripts into Microsoft Windows executables. It packages all required libraries and compilers into a compression file, so that users can run the Python script without installing the Python environment. We did try to use different packaging tools to see if we can obtain a different result. Unfortunately, by repeating those steps used in PyInstaller compilation, Py2Exe produced the same result as that of PyInstaller, ModuleNotFoundError: [!] Config for vits cannot be found, and we got stuck here again.
